additional download is required, https://drive.google.com/file/d/1JICbL_1BevMWTBNLowmjLNYCMND1Fx5n/view?usp=sharing must be placed in the ckpt folder before running predictor

to run the image dataset generator, the command 
pip install six numpy scipy Pillow matplotlib scikit-image opencv-python imageio Shapely
 can install most of the requirements

The following command can then be run to install the image augmenter module 

pip install imgaug

the following commands are also needed to be run to installl other required modules 

pip install lxml
pip install imageio
pip install Pillow



For running mainv2, the following modules are required:

darkflow https://github.com/thtrieu/darkflow
cython
opencv
tensorflow/tensorflow-gpu
